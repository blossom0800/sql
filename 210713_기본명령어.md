# SQL 기본 명령어 분류
  - 데이터 정의어(DDL), 데이터 조작어(DML), 데이터 제어어(DCL), 트랜젝션 제어어(TCL)

# 1) 데이터 정의어(DDL: Data Definition Language)
    - 테이블 생성, 변경, 삭제
# 2) 데이터 조작어(DML: Data Manipulation Language)
    - 데이터 삽입, 조회, 수정, 삭제
# 3) 데이터 제어어(DCL: Data Control Language)
    - 데이터 접근 권한 부여, 제거
# 4) 트랜젝션 제어어(TCL: Transaction Control Language)
    - 데이터 조작어(DML) 명령어 실행, 취소, 임시저장
    
    
# DBA, Data Analyst의 역할
    - DBA(DataBase Administrator) 역할: 데이터베이스 관리자, 기업 내에서 데이터베이스 관리
    - Data Analyst 역할: 데이터 분석을 통해 새로운 인사이트 도출
